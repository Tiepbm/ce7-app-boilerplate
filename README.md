## :arrow_forward: Usage

```sh
npx @react-native-community/cli init MyApp --template git@github.com:Tiepbm/ce7-app-boilerplate.git
```
## :arrow_forward: Rename

```sh
npx react-native-rename "CE7" -b "com.ce7.test"
```
